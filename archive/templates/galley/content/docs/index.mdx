# Welcome to Galley

Galley is Inkform's own design system for documentation — warm paper + ink, one
editorial "Proof" red-pencil accent, a serif prose voice beside a mono machine
voice. Powered by [`@inkform/framework`](https://github.com/inkform-dev/framework):
write Markdown, commit it, and ship a fast, searchable, themeable site — no CMS,
no database, no visual opinions in the way.

## Make it yours

Every theme is driven by the same engine and a single `theme.css` token file, so
you can recolor, re-font, and restyle the whole site without touching layout
code.
