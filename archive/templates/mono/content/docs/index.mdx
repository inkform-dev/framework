# MONO

Welcome to Mono — a monospace-first, terminal-styled docs starter powered by
[`@inkform/framework`](https://github.com/inkform-dev/framework).
Write Markdown, commit it, and ship a fast, searchable, no-database docs site
with a sharp brutalist identity and hot red-orange accent.

## Make it yours

Every theme is driven by the same engine and a single `theme.css` token file, so
you can recolor, re-font, and restyle the whole site without touching layout
code.
