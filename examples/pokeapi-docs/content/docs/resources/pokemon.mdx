# Pokémon

The `/pokemon/` endpoint returns the most comprehensive resource in PokéAPI. A single Pokémon object includes base stats, type assignments, learnable moves, held items, sprites, and links to the species resource for flavour text and evolution data.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/pokemon/` | List all Pokémon (paginated) |
| GET | `/pokemon/{name}` | Get a Pokémon by name |
| GET | `/pokemon/{id}` | Get a Pokémon by National Pokédex ID |

## Example request

<CodeGroup>
  ```bash curl
  curl https://pokeapi.co/api/v2/pokemon/gengar
  ```

  ```js fetch
  const res = await fetch('https://pokeapi.co/api/v2/pokemon/gengar');
  const gengar = await res.json();
  ```
</CodeGroup>

## Key fields

<ResponseField name="id" type="integer" required>
  The National Pokédex number. Stable and unique.
</ResponseField>

<ResponseField name="name" type="string" required>
  Lowercase hyphenated name. For forms: `pikachu-original-cap`, `deoxys-attack`.
</ResponseField>

<ResponseField name="base_experience" type="integer">
  Base XP yield when defeated. Used in experience calculations.
</ResponseField>

<ResponseField name="height" type="integer" required>
  Height in decimetres (divide by 10 for metres).
</ResponseField>

<ResponseField name="weight" type="integer" required>
  Weight in hectograms (divide by 10 for kilograms).
</ResponseField>

<ResponseField name="types" type="PokemonType[]" required>
  Type assignments. `slot` 1 is the primary type, `slot` 2 is the secondary (if any).
  Each entry has a `type` NamedAPIResource.
</ResponseField>

<ResponseField name="stats" type="PokemonStat[]" required>
  Base stats array. Stat names: `hp`, `attack`, `defense`, `special-attack`,
  `special-defense`, `speed`. `effort` is the EV yield.
</ResponseField>

<ResponseField name="abilities" type="PokemonAbility[]" required>
  All abilities. `is_hidden: true` marks the hidden ability. `slot` is 1, 2, or 3.
</ResponseField>

<ResponseField name="moves" type="PokemonMove[]" required>
  All learnable moves across all game versions. Each move entry includes
  `version_group_details` with how and at what level it's learned.
</ResponseField>

<ResponseField name="sprites" type="PokemonSprites" required>
  Image URLs. `front_default` is the most widely used.
  `other['official-artwork'].front_default` is the high-res artwork.
</ResponseField>

<ResponseField name="species" type="NamedAPIResource" required>
  Link to the Pokémon Species resource — contains flavour text, evolution chain,
  egg groups, and cross-generation data.
</ResponseField>

## Tips

<Accordion title="Getting flavour text (Pokédex entries)">
  Flavour text is NOT in the `/pokemon/` resource. Fetch the linked `species` URL:

  ```js
  const p = await fetch('https://pokeapi.co/api/v2/pokemon/gengar').then(r => r.json());
  const species = await fetch(p.species.url).then(r => r.json());
  const entry = species.flavor_text_entries.find(e => e.language.name === 'en');
  ```
</Accordion>

<Accordion title="Checking type effectiveness">
  Fetch the type URL from `types[0].type.url` to get `damage_relations` with
  `double_damage_from`, `half_damage_from`, and `no_damage_from` arrays.
</Accordion>

<Accordion title="Getting all forms (Alolan, Galarian, etc.)">
  Regional forms appear as separate Pokémon with hyphenated names:
  `vulpix-alola`, `meowth-galar`. They have the same species but different
  stats and types.
</Accordion>
