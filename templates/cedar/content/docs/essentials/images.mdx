# Images & Embeds

Add images, video, and rich media to bring your documentation to life.

## Images

Use Markdown image syntax:

```md
![A descriptive alt text](/path/to/image.jpg)
```

Wrap an image in a `<Frame>` to add a border and an optional caption:

<Frame caption="Images look great framed.">
  ![Landscape](/hero.svg)
</Frame>

<Note>
  Keep images under ~5 MB. For larger assets, host them on a CDN and reference
  the URL.
</Note>

## Video

Embed a YouTube or Loom video with the `<Embed>` component:

```mdx
<Embed provider="youtube" url="https://youtu.be/dQw4w9WgXcQ" />
```

## iframes

Embed any external page:

```mdx
<Embed provider="iframe" url="https://example.com/widget" />
```
