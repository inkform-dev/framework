# Introduction

Welcome to the home of your new documentation. This starter is a complete,
deployable docs site powered by [`@inkform/framework`](https://github.com/inkform-dev/framework):
write Markdown, commit it, and ship a fast, searchable, themeable site — no CMS,
no database.

## Make it yours

Every theme is driven by the same engine and a single `theme.css` token file, so
you can recolor, re-font, and restyle the whole site without touching layout
code.

<Note>
  New here? Start with the [Quickstart](/quickstart) — you'll have a site running

</Note>
